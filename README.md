# DSA Tracker

🌐 Live: https://maheshbabukona.github.io/DSA_Tracker/

A simple DSA practice tracker that runs entirely in the browser.  
No backend, no login — progress is saved using localStorage.

---

## Topics

Arrays, Binary Search, Strings, Recursion & Backtracking, Linked List,  
Stack & Queue, Binary Trees, BST, Heaps, Trie, Graph,  
Dynamic Programming, Greedy

---

## Structure

Each day includes:
- 1 Easy  
- 1 Medium  
- 1 Hard  

If a topic has fewer problems, mediums are used to balance it.

---

## Features

- Track progress per topic  
- Overall completion tracking  
- Daily problem sets  
- Progress saved automatically  
- Direct links to problems  

---

## How to use

1. Open the site  
2. Solve problems  
3. Mark them as completed  

Progress is stored in your browser.

---

## Tech

- HTML, CSS, JavaScript  
- localStorage for persistence  

---

## Notes

- Works best on modern browsers  
- Data is stored locally (no sync across devices)

---
 
## How it works
 
Each day has **1 Easy + 1 Medium + 1 Hard**. When the ratios don't balance across a topic, mediums fill the gap. Every problem links directly to the problem page. Check it off when done — progress persists in `localStorage`.
 
- Per-topic progress bars
- Global completion tracker in the sidebar
- Day cards show live completion count
- Completed problems strike through and fade
